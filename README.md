# libreria_laboratorio-UNIR

codigo fuente para un administrador de libreria

Comandos utiles de maven:

- Compilar y empaquetar: `mvn package`
- Ejecutar desde el JAR (si no tienes dependencias externas): `java -cp target/mi-proyecto-1.0-SNAPSHOT.jar com.example.App`
- Ejecutar JAR ejecutable (incluye dependencias): `java -jar target/mi-proyecto-1.0-SNAPSHOT.jar`

 - Ejecutar desde el JAR (si no tienes dependencias externas): `java -cp target/mi-proyecto-1.0-SNAPSHOT.jar com.libreria.App`
 - Ejecutar JAR ejecutable (incluye dependencias): `java -jar target/mi-proyecto-1.0-SNAPSHOT.jar`

Requisitos: tener `maven` y `java` (JDK 17+) instalados.
